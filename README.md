# Сайт-лендинг для кондитерской "Сладкий Сундук"
Это полноценный сайт, который имеет функциональный back-end и front-end, а также рабочую версию задеплоиную на [Render](https://landing-sweet-chest.onrender.com/).
## Установка и запуск
Для запуска проекта нужна версия Python 3.11.9. 

Расспакуйте архив с кодом в папку. Далее запустите коммандную строку и введите эту комманду:
```
python -r requirements.txt 
```

Её нужно выполнить в директории с файлами проекта.

Далее нужен файл .env(Создайте его в корне проекта). Его содержимое должно быть таким:

```
RESEND_API_KEY="Апи ключ от сервиса Resend"
EMAIL_FROM=onboarding@resend.dev(Не менять)
EMAIL_TO="Ваша почта"
PORT="Порт сайта(обычно 5000)"
```
[Ссылка на сервис Resend](https://resend.com/)

После создание .env файла нужно заполнить таблицу(файл cards.xlsx):

| A  | B | C  | D |
| --- | --- | --- | --- |
| CardName  | CardIngredients | CardImage | CardPrice |
| Имя товара  | Ингредиенты | Картинка Товара(поместить её в папку static/Media) | Цена |

Дальше хостим сайт:
```
python main.py
```

Все готово!Для того чтобы увидеть страницу нужно перейти по [этому](http://127.0.0.1:5000) адресу.
